# 🐶 Cafofo do Paçoca – Hotel Canino

Bem-vindo ao **Template de Landing Page do Cafofo do Paçoca**, um hotel canino pensado com carinho para apresentar os serviços de hospedagem, cuidados e bem-estar de pets com sofisticação, acolhimento e presença digital.

Este modelo é ideal para **pet shops, hotéis caninos, cuidadores independentes** ou qualquer serviço especializado em **cuidados premium para cães**, incluindo hospedagem, banho, tosa, recreação e adestramento.

Desenvolvido com um visual moderno, responsivo e cativante, este projeto transmite **confiança, profissionalismo e amor pelos animais**. Perfeito para atrair tutores, agendar serviços ou apresentar seu espaço, equipe e diferenciais.

---

## 🐾 Ideal para:

- Hotéis caninos e creches para pets  
- Pet shops com serviços de hospedagem  
- Profissionais de pet care (banho, tosa, adestramento)  
- Serviços premium para pets

📲 **Transforme visitantes em clientes fiéis** com uma apresentação visual encantadora, focada no carinho e cuidado que seu negócio oferece.

# Como subir as alterações com Git

1. Abra o terminal na pasta do seu projeto.
2. Execute os comandos abaixo:

```sh
git add .
git commit -m "Ajustes no menu mobile e responsividade"
git push
```

- `git add .` adiciona todas as alterações.
- `git commit -m "mensagem"` cria um commit com uma mensagem.
- `git push` envia para o repositório remoto.

Se for o primeiro push, pode ser necessário configurar o repositório remoto com:
